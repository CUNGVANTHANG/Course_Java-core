## 1. Mảng 1 chiều (Array)

**Tạo mảng có giá trị:**
```java
int[] age = {24, 23, 18, 19, 20};
```

hoặc

```java
int numbers[] = {24, 23, 18, 19, 20};

```
**Tạo mảng bằng cách xác định kích thước:**
```java
int[] numbers = new int[4]; // 4 chính là số lượng phần tử trong mảng
```

hoặc

```java
int numbers[] = new int[4]; // 4 chính là số lượng phần tử trong mảng
```
